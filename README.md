# Masked Autoencoder Visualizer

This repository provides a visualizer for Masked Autoencoders (MAE). A Masked Autoencoder is a type of neural network architecture that is designed to perform representation learning by predicting masked portions of the input data. It operates by randomly masking parts of the input and then training the model to reconstruct the original input from the unmasked parts.

## Deployment

You can access the visualizer at: [MAE Visualizer](https://mae-visualizer.vercel.app)
